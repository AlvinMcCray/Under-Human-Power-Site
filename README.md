# Under Human Power - Expedition Website

Official website documenting Darrell J. Gardner's human-powered expedition from the Mexican border to the Arctic Circle.

## About the Journey

This expedition covered 5,700+ miles using only human power:
- **Hiking**: Mexican border to Northern California along the Pacific Coast Highway
- **Kayaking**: Through the Inside Passage of British Columbia
- **Hiking**: Final push through Yukon Territory to the Arctic Circle

Zero motors. One man. Three disciplines.

## Website Structure

This is a 4-page static website:
- **Home** (`index.html`) - Hero section with image carousel and expedition overview
- **About** (`about.html`) - Darrell's bio and contact information
- **Sponsors** (`sponsors.html`) - Expedition supporters and partners
- **Journey** (`journey.html`) - Route map, timeline, and SPOT tracking data

## Tech Stack

- Pure HTML/CSS/JavaScript (no frameworks)
- Responsive design
- Image carousel with auto-cycling (6 second intervals)
- Color scheme: Orange (#ff6600), Black, White, Grey

## Local Development

1. Clone this repository
2. Open the project folder in VS Code
3. Install "Live Server" extension
4. Right-click `index.html` → "Open with Live Server"
5. Site opens at `http://127.0.0.1:5500`
